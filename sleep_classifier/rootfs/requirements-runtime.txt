# Runtime-only dependencies for the v1.3.0 add-on.
#
# Design notes
# ------------
# Since v1.3.0 the add-on no longer runs a local sleep-stage classifier —
# it subscribes to a stage entity the user has already built in HA.  The
# runtime image therefore needs none of the heavy scientific stack:
#
#   * no ``scipy`` / ``h5py`` / ``PyWavelets`` — those were only needed by
#     the deleted CNN/BiLSTM/wavelet pipeline.
#   * no ``paho-mqtt`` — the MQTT bridge modules were deleted too; we
#     talk to HA over the REST + WebSocket API exclusively.
#   * ``numpy`` stays because a handful of utilities (e.g. sleep-debt
#     computations) still rely on it, and it's ~2 MB so the savings
#     wouldn't justify a numpy-less refactor.
#
# Required invariant: ``import src`` must succeed using only the packages
# listed below.
aiohttp>=3.9.0
numpy>=1.24.0
