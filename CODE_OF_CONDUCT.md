## 我们的承诺

身为社区成员、贡献者及领导者，我们承诺让每个参与本社区的人都免受骚扰，
无论其年龄、体型、可见或不可见的身心障碍、族群、性别认同与表达、经验水平、
教育程度、社会经济地位、国籍、个人外表、种族、阶级、宗教信仰、性取向或
性别取向。

我们承诺以有助于建立开放、友善、多元、包容、健康社区的方式行事与互动。

## 我们的标准

有助于营造正向社区环境的行为示例：

- 表现出同理心与对他人的善意。
- 尊重不同的观点、立场与经验。
- 给予并优雅地接受建设性反馈。
- 主动承担责任、向受到影响的人致歉，并从经验中学习。
- 关注的不仅是个人，更是整个社区的整体利益。

无法接受的行为示例：

- 使用性化的语言或图像，以及任何形式的性关注或性挑逗。
- 侮辱或诋毁性的言论，以及人身或政治攻击。
- 公开或私下的骚扰行为。
- 未经明确许可，公开他人的私人信息（例如住址或电子邮件）。
- 在专业场合下，其它可被合理认定为不当的行为。

## 执行责任

社区领导者负责阐明并执行可接受行为的标准，并将对其认为不当、有威胁、冒犯
或有害的任何行为采取适当且公正的纠正措施。

社区领导者有权利且有责任删除、编辑或拒绝任何不符合本行为准则的评论、提交、
代码、wiki 编辑、issue 及其它贡献，并在必要时说明缘由。

## 适用范围

本行为准则适用于所有社区场合，亦适用于个人在公共空间代表本社区的情形。
代表本社区的示例包括使用官方电子邮件地址、通过官方社交媒体账号发文，或
在线上或线下活动中担任受指派的代表。

具体到本仓库，「社区场合」包括但不限于：

- GitHub 仓库 [LiangyuLu-lly/HA-sleep](https://github.com/LiangyuLu-lly/HA-sleep)
  的 Issues、Pull Requests、Discussions、Wiki、Security Advisories。
- 与本项目相关的任何官方频道（如未来可能创建的 Matrix / Discord / 邮件列表）。
- 项目维护者代表本项目参与的外部社区活动（HA 社区论坛、Reddit
  r/homeautomation、HN、知乎专栏等）。

## 执行准则

社区领导者将依据下列准则判定违反本行为准则之行为的后果：

### 1. 纠正

**社区影响**：使用了不当言语，或在社区中被认为是不专业、不受欢迎的其它行为。

**后果**：社区领导者发出私下、书面的警告，明确说明违规行为的性质，并解释
为何该行为不当。可能要求公开道歉。

### 2. 警告

**社区影响**：通过单一事件或一系列行为造成违规。

**后果**：警告并附带对持续行为的处置后果。在指定时间内，不得与相关人员互动，
包括与执行行为准则人员的非主动互动。这包括避免在社区场合以及外部渠道（如
社交媒体）的互动。违反这些条款可能导致暂时或永久封禁。

### 3. 暂时封禁

**社区影响**：严重违反社区标准，包括持续性的不当行为。

**后果**：在指定时间内，暂时禁止与社区进行任何形式的互动或公开沟通。
在此期间，不得与相关人员有任何公开或私下的互动，包括与执行行为准则人员的
非主动互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、对个人的骚扰、
或对某类人的攻击或贬低。

**后果**：永久禁止在社区内进行任何形式的公开互动。
